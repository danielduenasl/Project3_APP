## Aplicación de creacion de eventos "Party Planner"

Bienvenidos a la aplicación *Party Planner*, esta aplicación ayuda a crear eventos según el deseo de los usuarios, esta aplicación permite que, los usuarios ingresen el nombre de un evento, la ubicación, el horario, que agreguen personas de sus contactos ya sea los invitados y que los invitados queden guardados, esto en caso el usuario quiere crear otro evento en un futuro, solo agregue los invitados que ya estan guardados en los futuros eventos. Si en algún dado caso no hay alguna personas agregada en la lista de contactos, se tiene la opción de agregar a más personas.
